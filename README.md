# 📊 Data Science & Analytics Job Market Analysis

This project presents an end-to-end analysis of the **Data Science** and **Data Analytics** job market using data collected from Naukri.com. Multiple datasets were merged, cleaned, and transformed into a unified analytical dataset before being visualized through an interactive Power BI dashboard.

The project demonstrates the complete data analytics workflow—from data preprocessing and feature engineering to dashboard development and business insight generation.

---

## 📌 Project Overview

The objective of this project is to analyze hiring trends for Data Science and Data Analytics roles by examining salaries, experience requirements, skills, hiring companies, and job locations.

Using Python for data preparation and Power BI for visualization, the project provides actionable insights into the current job market.
